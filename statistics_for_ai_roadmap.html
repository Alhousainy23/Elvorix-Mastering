Descriptive Statistics — ليه مهمة في AI؟
Mean (المتوسط): بيحسب متوسط الـ loss أو accuracy على الـ batch
Variance / Std: بيكشف إذا كان الـ model بيتذبذب (unstable training)
Median: أقل تأثرًا بالـ outliers — مهم في تنظيف البيانات
import numpy as np data = [2, 4, 4, 4, 5, 5, 7, 9] print("Mean :", np.mean(data)) # 5.0 print("Std :", np.std(data)) # 1.85 print("Variance:", np.var(data)) # 3.43 print("Median :", np.median(data)) # 4.5
Probability — الاحتمالات في AI
Bayes' Theorem: أساس Naive Bayes classifier وكمان Bayesian Neural Networks
Conditional Probability P(A|B): جوهر كل خطوة في الـ deep learning
Joint Probability: مستخدمة في Generative Models (VAE, GAN)
from fractions import Fraction # Bayes: P(cat|meows) = P(meows|cat) * P(cat) / P(meows) P_cat = 0.4 # prior P_meow_cat= 0.9 # likelihood P_meow = 0.5 # evidence P_cat_meow = (P_meow_cat * P_cat) / P_meow print(f"P(cat | meows) = {P_cat_meow:.2f}") # 0.72

Hypothesis Testing — في AI بستخدمها فين؟
A/B Testing: مقارنة نموذجين — هل الـ accuracy فرقها significant؟
p-value: إذا p < 0.05 الفرق حقيقي مش عشوائي
Confidence Interval: بيحدد مدى الثقة في الـ model predictions
from scipy import stats import numpy as np model_A = np.array([0.82,0.85,0.80,0.83,0.84]) model_B = np.array([0.88,0.90,0.87,0.91,0.89]) t_stat, p_value = stats.ttest_ind(model_A, model_B) print(f"p-value = {p_value:.4f}") if p_value < 0.05: print("Model B أفضل بشكل معنوي ✓")

Optimization — الرياضيات وراء التدريب
Gradient Descent: المشتقة بتقولك اتجاه التحسين — w = w - lr × ∂Loss/∂w
Adam Optimizer: بيجمع Momentum + Adaptive LR — الأشهر في Deep Learning
Learning Rate: أهم hyperparameter — كبير = unstable، صغير = بطيء
import numpy as np # Gradient Descent يدوي def loss(w): return (w - 3)**2 # minimum عند w=3 def grad(w): return 2*(w - 3) w, lr = 0.0, 0.1 for step in range(10): w = w - lr * grad(w) print(f"Step {step+1}: w={w:.3f}, loss={loss(w):.4f}")
Regularization — علاج الـ Overfitting
L2 (Ridge): بيضيف عقوبة على الـ weights الكبيرة — الأشيع في الـ Neural Networks
Dropout: بيطفّي neurons عشوائياً أثناء التدريب — بيمنع الاعتماد المفرط
Early Stopping: بيوقف التدريب لما الـ validation loss يبدأ يزيد
import torch.nn as nn class RegularizedNet(nn.Module): def __init__(self): super().__init__() self.fc1 = nn.Linear(128, 64) self.drop = nn.Dropout(p=0.3) # 30% neurons off self.fc2 = nn.Linear(64, 10) def forward(self, x): x = self.fc1(x).relu() x = self.drop(x) # Dropout هنا return self.fc2(x) # L2 Regularization عبر weight_decay في الـ optimizer # optimizer = torch.optim.Adam(model.parameters(), weight_decay=1e-4)
Evaluation Metrics — إزاي تحكم على الـ Model
Precision: من اللي قلت إنه positive، كام صح؟ — مهم لما False Positive غالي
Recall: من الـ positive الحقيقيين، قدرت تمسك كام؟ — مهم في medical diagnosis
AUC-ROC: بيقيس الـ model على كل الـ thresholds — 0.5 = random، 1.0 = perfect
from sklearn.metrics import classification_report, roc_auc_score y_true = [0,1,1,0,1,0,1,1] y_pred = [0,1,0,0,1,0,1,1] y_prob = [.1,.9,.4,.2,.8,.3,.7,.95] print(classification_report(y_true, y_pred)) print(f"AUC-ROC = {roc_auc_score(y_true, y_prob):.3f}")
